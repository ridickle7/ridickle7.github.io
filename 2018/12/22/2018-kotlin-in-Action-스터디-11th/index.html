not only 개별 클래스 집중 but also 타 클래스와 상호작용하는 연결지점 (인터페이스) 확인 필요

영역 특화 언어라는 개념

DSL

  • 영역 특화 언어
  • 특정 영역에서 사용하는 어휘, 문법, 의미론을 사용
  • 예전부터 있던 개념이나, 도메인 특화 모델링의 수요증가로 더 알려짐

  • DSL example : SQL(데이터베이스), 정규식(문자열)
  • 클래스나 함수 선언의 필요 없이 즉시 활용 가능 (압축적인 문법)
